type: feedback

Writing Material 飞书文档每个 Panel 的固定槽位（按顺序）：
1. **一句话结论** + **图文件路径**
2. **详细中文结果段落**：引用具体数字，面向毕业论文正文，遵循观察→比较→解释→边界的逻辑
3. **PPT 页内文字（英文）**：一句 take-home title + 2-4 bullets，面向答辩/组会
4. **子图图例（中文）**：说明子图展示什么、颜色/轴/分组含义，以"（A）""（B）"格式

文档级内容（放末尾）：
5. **论文段落草稿**：精炼学术段落，串联多个 Panel 成连续论证
6. **整张 Figure 总图例（中文）**：总标题 + 按（A）（B）（C）依次概括

文档开头必须有 **Analysis metadata**（model, evaluation mode, dataset split, primary metric 等）。

**Why:** 用户同时需要毕业论文正文、PPT 汇报、图注、SCI 投稿等多种输出格式。旧的 3 层结构缺少子图图例和总图例。

**How to apply:** 创建 Writing Material 文档前先做 panel-level audit（提取一句话结论、检查编号/路径一致性、列出 TODO），再按固定槽位输出。学术措辞避免过强表述（"证明""必然"），用"提示""表明""与…一致"。
